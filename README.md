# Portfolio — Ezequiel Rifos

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

🔗 **[Live Demo](https://portfolio-renzo-rifos.vercel.app)**

## Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, Locomotive Scroll
- **Contact:** EmailJS
- **Deploy:** Vercel

## Features

- Responsive design across all screen sizes
- Smooth scroll animations
- Interactive tech belt with SVG icons
- Contact form with EmailJS integration
- Dark theme with custom color palette

## Run locally

```bash
# Clone the repo
git clone https://github.com/Renzo-rifos/portfolio.git
cd portfolio

# Install dependencies
pnpm install

# Start dev server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
